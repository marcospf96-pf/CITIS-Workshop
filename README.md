# CITIS-Workshop

##  Configurar el ambiente

### Identificar NIC para conexión inalambrica

```
ifconfig
```

### Actualizar Linux

```
sudo apt update
```

### Collect Repository

```
git clone https://github.com/v1s1t0r1sh3r3/airgeddon.git
cd airgeddon
ls
```

### Run script

```
./airgeddon.sh
```
